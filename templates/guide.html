User Guide

Smart Door Lock System User Guide

1. Dashboard

The dashboard provides quick access to the main functions of the system:

  • Door Control: Lock or unlock the door remotely.
  • Camera Control: Turn the livestream on/off, capture images, and view recent captures.
  • Recent Activity: View the most recent door activities and user logins.

2. Settings

Configure system settings:

  • System Passcode: The main passcode used to unlock the door.
  • Max Trials: Maximum number of failed attempts before lockout.
  • Lockout Passcode: Special passcode to override lockout.

3. People Management

Manage users of the system (Admin only):

  • Add Users: Create new user accounts.
  • Edit Users: Modify existing user information and permissions.
  • Delete Users: Remove users from the system.

4. My Profile

View and edit your own profile information:

  • Update Information: Change your name, email, and password.
  • View Permissions: See what actions you are authorized to perform.

5. Metrics

View system statistics and performance metrics:

  • Authentication Stats: Successful and failed login attempts.
  • Door Access Stats: Successful and failed door unlock attempts.
  • System Performance: CPU, RAM, temperature, and uptime.
  • Camera Performance: Camera latency and other metrics.

6. Activity Logs

Detailed logs of all system activities:

  • Filter Logs: View specific types of activities.
  • Download Logs: Export logs to CSV for analysis.

7. Notifications

Stay informed about important system events:

  • Real-time Alerts: Receive notifications about system events.
  • Notification Panel: Access all notifications from the bell icon in the header.

8. User Roles

The system has different user roles with varying permissions:

  • Admin: Full access to all system features.
  • Administrator: Similar to admin but cannot delete admin accounts.
  • Family Member: Can unlock doors and view logs.
  • Guest: Limited access, typically only door unlock.

9. Access Types

Users can have different types of access:

  • Full Access: No time restrictions.
  • Limited Access: Access expires after a specific date.
